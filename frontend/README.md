# DocuMind 前端项目

企业级文档智能处理系统前端看板，基于 Vue 3 + TypeScript + Vite 构建。

## UI 设计风格

### 玻璃拟态 (Glassmorphism) 设计

本系统采用现代化的玻璃拟态设计风格，核心视觉元素包括：

**核心 CSS 变量 (`variables.css`)：**

```css
/* 玻璃拟态效果 */
--glass-bg: rgba(255, 255, 255, 0.25);      /* 半透明背景 */
--glass-border: rgba(255, 255, 255, 0.18);   /* 微透边框 */
--glass-shadow: 0 8px 32px rgba(31, 38, 135, 0.15); /* 柔和阴影 */
--glass-blur: 10px;                           /* 背景模糊 */

/* 主色调 */
--color-primary: #1A73E8;                     /* 蓝色主色 */
--color-primary-light: #4FC3F7;              /* 浅蓝点缀 */
--color-success: #10B981;                     /* 成功状态绿 */
--color-warning: #F59E0B;                     /* 警告状态黄 */
--color-error: #EF4444;                       /* 错误状态红 */

/* 渐变效果 */
--gradient-primary: linear-gradient(135deg, #1A73E8, #4FC3F7);
--gradient-success: linear-gradient(135deg, #10B981, #34D399);
--gradient-bg: linear-gradient(180deg, #F8FAFC, #E2E8F0);
--gradient-sidebar: linear-gradient(180deg, #1E293B, #334155);
```

**玻璃卡片组件 (`global.css`)：**

所有卡片式组件都使用 `.glass-card` 类：

```css
.glass-card {
  background: var(--glass-bg);
  backdrop-filter: blur(var(--glass-blur));    /* 背景模糊 */
  border: 1px solid var(--glass-border);
  border-radius: var(--radius-lg);
  box-shadow: var(--glass-shadow);
  transition: all var(--transition-base);
}

.glass-card:hover {
  background: rgba(255, 255, 255, 0.35);
  box-shadow: var(--shadow-lg);
}
```

**动画效果：**

- `ripple-expand`：波纹扫描动画（文件上传时）
- `pulse-glow`：脉冲发光效果（AI 处理时）
- `fade-in`：渐入动画（页面加载时）

## 页面布局结构

### 整体布局

采用经典的侧边栏 + 顶栏 + 内容区三栏布局：

```
┌─────────────┬─────────────────────────────────────────┐
│   Sidebar   │                 Header                   │
│   (240px)   │              (搜索栏 + 用户操作)           │
│             ├─────────────────────────────────────────┤
│  导航菜单    │                                         │
│             │              Content Area                │
│  存储进度    │              (页面内容区域)               │
│             │                                         │
│  用户信息    │                                         │
└─────────────┴─────────────────────────────────────────┘
```

### 侧边栏 (Sidebar)

**位置**：`src/components/layout/Sidebar.vue`

**功能**：
- Logo 展示（DocuMind + 文档图标）
- 5 个导航菜单项：
  | 图标 | 名称 | 路径 |
  |------|------|------|
  | 📊 | 工作台 | `/dashboard` |
  | 📁 | 文件中心 | `/files` |
  | 🧪 | AI实验室 | `/ai` |
  | 📜 | 文档版本 | `/documents` |
  | ⚙️ | 系统设置 | `/settings` |
- 存储空间进度条（65% 示例）
- 用户信息展示（头像 + 用户名 + 角色）

**视觉特点**：
- 深色渐变背景 (`--gradient-sidebar`)
- 导航项悬停时背景变亮
- 激活项使用主色渐变高亮

### 顶部栏 (Header)

**位置**：`src/components/layout/Header.vue`

**功能**：
- 全局搜索栏（支持 Ctrl+K 快捷键聚焦）
- 搜索内容会跳转到文件中心并传递搜索参数
- 显示当前用户名
- 退出登录按钮（红色样式）

**视觉特点**：
- 玻璃卡片效果
- 搜索栏半透明背景
- 退出按钮悬停时变红色

### 6. AI实验室 (`/ai`)

**位置**：`src/pages/ai/index.vue`

**界面布局**：
```
┌─────────────────────────────────────────────────────────┐
│  功能标签栏                                              │
│  [🤖 RAG问答] [🔍 智能搜索] [⚡ 文档处理] [📚 知识库]       │
├─────────────────────────────────────────────────────────┤
│  根据选中标签显示对应面板内容                              │
└─────────────────────────────────────────────────────────┘
```

**四大功能模块**：

#### 6.1 RAG问答 (`rag` 标签)

**功能**：基于文档知识库进行智能问答

**界面元素**：
- 问题输入文本域
- 检索策略选择：
  | 策略 | 说明 |
  |------|------|
  | 默认 | 基础检索 |
  | BM25 融合 | BM25_FUSION |
  | 交叉编码器 | CROSS_ENCODER |
  | 混合策略 | HYBRID |
- 查询按钮

**结果展示**：
- 📝 答案区域（绿色边框卡片）
- 📚 相关上下文（如有）
- 📖 来源文档列表（如有）

#### 6.2 智能搜索 (`search` 标签)

**功能**：向量检索 + BM25 混合搜索

**界面元素**：
- 搜索关键词输入框
- 返回数量选择：5条 / 10条 / 20条
- 搜索按钮

**结果展示**：
- 每条结果显示：
  - 相关度得分（蓝色圆形徽章）
  - 文档内容片段
  - 文档ID（如有）

#### 6.3 文档处理 (`process` 标签)

**功能**：摘要、关键词、文本分析

**界面元素**：
- 文档内容输入文本域
- 处理选项复选框：
  - ☑️ 生成摘要
  - ☑️ 提取关键词
  - ☑️ 文本分析
- 处理按钮

**结果展示**：
- **摘要卡片**：
  - 摘要内容
  - 统计：原文字数 → 摘要字数 → 压缩率
- **关键词卡片**：
  - 关键词标签列表（带评分）
- **文本分析卡片**：
  - 6项统计数据：总字符、中文、英文、数字、句子数、行数

#### 6.4 知识库管理 (`kb` 标签)

**功能**：管理 RAG 知识库文档索引

**界面元素**：
- 🔄 刷新列表按钮
- 已索引文档列表（显示文档ID）
- 每个文档可🗑️删除
- 添加文档索引：
  - 文档ID输入框
  - 文档内容文本域
  - 📥 索引按钮

前端需要连接后端 API 服务：

- 默认地址：`http://localhost:8080`
- 确保后端服务已启动
- 后端启动：`docker-compose -f docker-compose-standalone.yml up -d`

## 开发指南

### 添加新页面

1. 在 `src/pages/` 创建页面组件
2. 在 `src/router/index.ts` 添加路由
3. 在 `src/components/layout/Sidebar.vue` 添加导航项

### 添加新API

1. 在 `src/types/api.ts` 定义类型
2. 在 `src/api/` 创建API模块
3. 在组件中引入使用

## 常见问题

### 端口冲突
如果 5173 端口被占用，Vite 会自动切换到下一个可用端口。

### 后端连接失败
检查：
1. 后端服务是否启动
2. `.env.development` 配置是否正确
3. 网络防火墙是否阻止

### Token刷新失败
- 确保 refreshToken 有效（7天有效期）
- 清除浏览器 localStorage 重新登录
